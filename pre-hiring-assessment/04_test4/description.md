# test4

## 문제

숫자를 입력 받으면 각 자릿의 숫자들을 곱한 결과물들이 하나의 숫자가 될 때까지 반복적으로 곱한 후, 최후의 한 자리 숫자를 리턴해야 합니다.

## 입력

### 인자 1 : num

- `number` 타입의 숫자 (num >= 0)

## 출력

- `number` 타입의 한 자리 숫자

## 입출력 예시

```javascript
let output = test4(786)
console.log(output); // --> 0
```

## 힌트

- 예를 들어 786을 인자로 넘길 경우, 함수는 0을 리턴해야 합니다.
  7 * 8 * 6 -> 336
  3 * 3 * 6 -> 54
  5 * 4 -> 20
  2 * 0 -> 0
