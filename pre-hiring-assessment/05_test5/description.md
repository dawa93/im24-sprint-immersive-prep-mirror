# test5

## 문제

수를 입력받아 각 자릿수를 모두 더한 값을 리턴해야 합니다.

## 입력

### 인자 1 : num

- `number` 타입의 정수

## 출력

- `number` 타입을 리턴해야 합니다.

## 주의 사항

- 음수를 입력받은 경우, 첫번째 수는 음수로 계산합니다.

## 입출력 예시

```javascript
let output = test5(1148);
console.log(output); // --> 14

output = test5(-316);
console.log(output); // --> 4
```
